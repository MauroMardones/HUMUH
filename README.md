# HUMUH — Spatio-temporal analysis of fishing effort multi-fleet in the Gulf of Cádiz

**Instituto Español de Oceanografía (IEO-CSIC)**  

---

## Context

This project analyses the spatio-temporal patterns of fishing fleet activity in the Gulf of Cádiz using two complementary data sources provided by the Junta de Andalucía:

- **VMS data** (*cajas verdes*, GPS-GPRS): vessel positioning records transmitted at regular intervals, containing coordinates, instantaneous speed, heading, and timestamp for each vessel.
- **Sales notes** (*Notas de Venta*): first-sale administrative records linking each fishing trip to the declared gear, species composition, weight, and economic value.

The combination of both sources allows assignment of a fishing gear to each VMS ping and derivation of spatially explicit fishing effort (hours fished) at a 0.05° × 0.05° grid (~5.6 km) across the study area.

## Pipeline execution order

Scripts in `R/` must be run sequentially. Each script saves its main output to `outputs/` as a `.fst` file, which the next script loads.

```
01 → 02 → 03 → 04 → 05
                  ↓
             06  07  08
```

Scripts 05–08 all read `outputs/esfuerzo_2018_2024.fst` independently, so they can be re-run in any order after script 04.

**To resume from an intermediate step** (new R session):

```r
library(data.table); library(fst)
esfuerzo <- read_fst("outputs/esfuerzo_2018_2024.fst", as.data.table = TRUE)
```

## Key analytical decisions

| Decision | Value | Rationale |
|---|---|---|
| Grid resolution | 0.05° (~5.6 km) | ICES WGSFD standard for small-scale fleets |
| Fishing state | speed filter [v_min, v_max] per gear | Standard VMS approach (Lee et al. 2010) |
| Effort metric | fishing hours (Σ Δt) | Consistent with logbook-VMS integration |
| CFG thresholds | 75% and 90% cumulative effort | Standard in spatial footprint studies |
| GLM family | Gaussian, identity link | Nested year within cell (Pennino et al. 2016) |
| Min. years for GLM | N_min = 3 | Avoids spurious trends with sparse data |
| Depth filter (COG) | prof_m ≥ 5 m | Excludes port pings and intertidal areas |
